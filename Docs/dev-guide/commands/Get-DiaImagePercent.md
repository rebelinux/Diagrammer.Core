# Get-DiaImagePercent

## SYNOPSIS

Used by As Built Report to get base64 image percentage calculated from image width.

## SYNTAX

### __AllParameterSets

```
Get-DiaImagePercent [[-GraphObj] <string>] [[-ImageInput] <string>] [-Percent <int>]
 [<CommonParameters>]
```
